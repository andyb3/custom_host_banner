*insert will insert banner before page content. This ensures page content is not covered, but it may not work on all sites.
on top will overlay banner on top of page content. This should work on more sites but some page content will be covered, so the banner will move out the way when the mouse moves over it.
Regex allows you to use regular expressions to match hosts. For example, ".*\.google\.com" would match all Google subdomains.
